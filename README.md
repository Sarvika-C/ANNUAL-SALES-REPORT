# 📊 Annual Sales Report Dashboard (Microsoft Excel)

An interactive **Annual Sales Report Dashboard** built in **Microsoft Excel** to analyze yearly sales performance and convert raw data into actionable business insights. The dashboard uses Pivot Tables, Pivot Charts, Slicers, and Excel visualization techniques to monitor revenue, orders, product performance, customer behavior, and regional sales trends.

---

## 📌 Project Overview

The **Annual Sales Report Dashboard** provides a comprehensive overview of yearly sales performance through interactive visualizations and dynamic filters. It enables users to track key performance indicators (KPIs), identify sales trends, compare product categories, evaluate regional performance, and support data-driven business decisions.
